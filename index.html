Description

VS Code est un éditeur de code

Créer un document

  1. Create a new folder on my computer
  2. Open VS Code
  3. Drag and drop the folder in VS Code so it appears on the left
  4. Create a file : “index.html” (have to name it like that, you don't have any choice)
  5. Use "emmet html 5"
